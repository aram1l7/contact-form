This app is made with create-next-app, material-ui and nodemailer

Installation

yarn install
yarn dev
The code will start runing on localhost:3000

In order to send mails with your own nodemailer account you need to
create app password in your google account.

After creating an app password

Create a .env file in the root of the project.

Add the following environment variables in the .env file:

NODEMAILER_USER= your gmail address
NODEMAILER_PASS= your app password
